Note: This section does not appear in the original article.

The author neglected to specify an authority for the binomial name Jaculus labradorius.

Given the description the author gives, he is almost certainly referencing Dipus labradorius [Kerr, 1792; TSN 952140] which is now considered to be a junior synonym of Zapus hudsonius [Zimmermann, 1780]. Several subspecies are described, Illinois falls within the range of the nominate subspecies Z. h. hudsonius [Zimmermann, 1780].

This species is currently referred to using the common name ‘Meadow Jumping Mouse’.

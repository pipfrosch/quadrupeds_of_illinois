Note: This section does not appear in the original article.

The author neglected to specify an authority for the binomial name Arvicola scalopsoides.

Given both the specific name and the description the author gives, he is almost certainly referencing Arvicola scalopsoides [Audubon and Bachman, 1841; TSN 971223] which is now considered to be a junior synonym of Microtus pinetorum [Le Conte, 1830]. Several subspecies are described, Illinois falls within the range of the subspecies M. p. scalopsoides [Audubon and Bachman, 1841] although many no longer recognize the validity of any subspecies within M. pinetorum.

This species is currently referred to using the common name ‘Woodland Vole’.
